Tyk Gateway Docker
=================================

This container only contains the Tyk OSS API Gateway, the Tyk Dashboard is provided as a separate container and needs to be configured separately.


# Installation


## Docker-Compose

With docker-compose, simply run 
```
$ docker-compose up -d
```
